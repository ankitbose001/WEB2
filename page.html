Widgets UI design in Flutter typically involves assembling and/or creating various widgets. A widget in Flutter represents an immutable description of part of the user interface; all graphics, including text, shapes, and animations are created using widgets. More complex widgets can be created by combining many simpler ones. However, the use of widgets is not strictly required to build Flutter apps. An alternative option is to use the Foundation library's methods directly, interfacing with "canvas" commands to draw shapes, text, and imagery directly to the screen. This property of Flutter has been utilized in a few frameworks, such as the open-source Flame game engine.
